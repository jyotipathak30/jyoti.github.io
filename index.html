HTML
HTML stands for Hyper Text Markup Language, which is the most widely used language on Web to develop web pages.
CSS
CSS stands for Cascading Style Sheets. CSS is used to control the style of a web document in a simple and easy way.
JS
JavaScript is a lightweight, interpreted programming language. It is designed for creating network-centric applications. It is open and cross-platform.
PHP
PHP stands for Hypertext Processor. It is a server side scripting language that is used to produce dynamic web content among other things.
BOOTSTRAP
Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development.
ANGULAR
Angular is a platform and framework for building single-page client applications using HTML and TypeScript.
SEO
SEO stands for “search engine optimization.” It’s the practice of increasing both the quality and quantity of website traffic, as well as exposure to your brand, through non-paid (also known as "organic") search engine results.
HOSTING
Hosting is a service through which storage and computing resources are providing to an individual or organization for the accommodation and maintenance of one or more websites and related services.
